## 변수
```JavaScript
var fruit; //변수 선언

fruit = "apple"; // 변수 초기화
```

## 변수 생성 시 주의 사항
* 변수명은 숫자로 시작할 수 없음
* 변수명은 최대한 자세하게 작성
* 의미가 불명확한 단어들의 조합은 피해야 함
