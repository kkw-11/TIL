# Linux
    * 인델 80386 이상의 프로세서를 사용하는 PC에서 운영되는 유닉스 운영체제의 공개 버전
# Linux 특징
* 오픈소스
    * 무료이고, 시스템의 특성에 맞게 소스를 변경할 수 있음
* 가상메모리와 공유 라이브러리
    * 하드드라이브를 가상 메모리로서 사용할 수 있음
        * 실제의 RAM 용량을 확장시킴
        * 공유 라이브러리를 구현함
        * 각각의 응용 프로그램이 공통되는 루틴을 별도로 저장하지 않아서 시스템의 많은 하드 공간을 줄여줌
* 저사양에서도 강력한 성능을 발휘함
* 프로그램의 호환성이 떨어짐
