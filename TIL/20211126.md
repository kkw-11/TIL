# python 가상환경
아래의 명령어로 실행하면 example이라는 폴더가 생기고 파이썬 가상환경을 생성한다.

```
python -m venv example
```

아래의 명령어를 실행하면 exaple폴더안에 Scripts안에 activate를 실행해서 가상환경으로 들어간다.
```
source example/Scripts/activate
```
